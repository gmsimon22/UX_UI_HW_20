Introduction

NSA Redesign

As designers, our team's goal was to redesign the NSA desktop and mobile websites to match their goals. These goals included improving usability and transforming a content-heavy, overwhelming site into a welcoming, simplified one.

The NSA mainly provides cryptology and signals intelligence to the US Government.

Research

Proto Persona

Our Protopersona depicts a man who is sceptical about government security and searches on the site to learn more.

User Path

By taking note of the user paths, we understood which path to focus our design sample on, as well as positives and negatives about the structure of the current design.

Usability Testing

We performed usability testing to understand the dilemmas that users are facing with the site. We organized the information to understand how to better configure our redesigns.

Site Map

Analyzing the paths on the site, we noted and organized the site paths with the goal of creating more simplistic, managable paths.

Original Designs

Using Figma, we each created original designs of the NSA website based on our user research.

User Tests

We performed user tests to evaluate the usability of our designs.

The Problem

The NSA aims to increase the transparency of their organization to citizens through this website. The current website design is chunky and heavily worded, making it overwhelming for users to find the information they need on the site. As designers, we will utilize our research to redesign the website into a more simplified, transparent website design.

Iteration

  • Style Tile
    The style tile I made creates a one-stop-shop for all necessary buttons, icons, fonts, and colors. This formed a reference for the design iterations.
  • Desktop Iteration
    For the desktop iteration, I adapted the design to be easier for the user to view. The User can see each section fully within the frame of the screen. The style is original and fitting for the NSA website. Information is easy to find and laid out in a simple, digestible way.
  • Mobile Iteration
    The mobile iteration matches the desktop iteration, with larger font sizes for the titles and beveled buttons, creating a modern, clean appearance.
